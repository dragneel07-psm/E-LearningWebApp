# UX Review Checklist

## Role Journeys
- [ ] School admin critical flow reviewed
- [ ] Teacher critical flow reviewed
- [ ] Student critical flow reviewed
- [ ] Parent critical flow reviewed

## Interaction Quality
- [ ] Empty/loading/error states defined
- [ ] Form validation states defined
- [ ] Success/failure toasts/messages clear

## Accessibility
- [ ] Keyboard navigation works
- [ ] Dialogs have title/description and focus trap
- [ ] Color contrast reviewed for key screens
- [ ] ARIA roles/labels added where needed

## Responsive Behavior
- [ ] Mobile layout tested
- [ ] Tablet layout tested
- [ ] Desktop layout tested

## Hand-off
- [ ] Acceptance criteria linked to tickets
- [ ] Component usage documented
