# Security Hardening Template

## Access and Auth
- [ ] RBAC matrix updated.
- [ ] Least privilege review completed.
- [ ] Token/session policy validated.

## Tenant Isolation
- [ ] Cross-tenant read test passed.
- [ ] Cross-tenant write test passed.

## Edge Protection
- [ ] Rate limiting configured.
- [ ] WAF policy active.

## Data Protection
- [ ] Encryption in transit enforced.
- [ ] Sensitive data handling documented.

## Audit and Monitoring
- [ ] Audit logs cover critical actions.
- [ ] Security alerts routed to on-call.

## Recovery
- [ ] Backup restore drill completed.
