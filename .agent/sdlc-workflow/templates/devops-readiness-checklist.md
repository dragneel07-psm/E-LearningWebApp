# DevOps Readiness Checklist

## Repository and Branching
- [ ] Branch strategy documented.
- [ ] PR template enforced.
- [ ] Protected branch rules enabled.

## CI Pipeline
- [ ] Lint stage required.
- [ ] Test stage required.
- [ ] Build stage required.
- [ ] Artifact retention configured.

## Environments
- [ ] Dev environment stable.
- [ ] Staging environment mirrors prod behavior.
- [ ] Production deploy path documented.

## Secrets and Config
- [ ] Secret store defined.
- [ ] Rotation ownership assigned.
- [ ] Env var contract documented.

## Deployment Safety
- [ ] Rollback procedure tested.
- [ ] Migration safety checks in pipeline.
- [ ] Release approval gate configured.
