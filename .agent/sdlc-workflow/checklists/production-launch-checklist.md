# Production Launch Checklist

## Launch Readiness
- [ ] Release notes published.
- [ ] Deployment rollback command validated.
- [ ] DB migration plan reviewed.

## Monitoring
- [ ] Error tracking active.
- [ ] Alert routing verified.
- [ ] Dashboard links shared with on-call.

## Support
- [ ] Support SLA and escalation path active.
- [ ] Hypercare staffing scheduled.

## Recovery
- [ ] Backup status verified.
- [ ] Disaster recovery runbook reviewed.

## Final Approval
- [ ] Product owner
- [ ] Engineering owner
- [ ] Operations owner
