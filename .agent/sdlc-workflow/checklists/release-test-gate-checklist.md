# Release Test Gate Checklist

## Functional
- [ ] Auth flows pass for all roles.
- [ ] Tenant creation and isolation tests pass.
- [ ] Core LMS flows pass (course, materials, assessments, timetable).

## Technical
- [ ] Unit and integration suites pass.
- [ ] E2E critical suite passes.
- [ ] Performance sanity test passes.

## Security
- [ ] Permission boundary tests pass.
- [ ] No unresolved critical vulnerabilities.

## Sign-off
- [ ] QA lead approved.
- [ ] Engineering lead approved.
- [ ] Product approved.
